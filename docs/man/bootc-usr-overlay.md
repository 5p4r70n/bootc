# NAME

bootc-usr-overlay - Add a transient writable overlayfs on \`/usr\` that
will be discarded on reboot

# SYNOPSIS

**bootc-usr-overlay** \[**-h**\|**\--help**\] \[**-V**\|**\--version**\]

# DESCRIPTION

Add a transient writable overlayfs on \`/usr\` that will be discarded on
reboot

# OPTIONS

**-h**, **\--help**

:   Print help

**-V**, **\--version**

:   Print version

# VERSION

v0.1.0
