# NAME

bootc-edit - Change host specification

# SYNOPSIS

**bootc-edit** \[**-f**\|**\--filename**\] \[**\--quiet**\]
\[**-h**\|**\--help**\] \[**-V**\|**\--version**\]

# DESCRIPTION

Change host specification

# OPTIONS

**-f**, **\--filename**=*FILENAME*

:   Use filename to edit system specification

**\--quiet**

:   Dont display progress

**-h**, **\--help**

:   Print help

**-V**, **\--version**

:   Print version

# VERSION

v0.1.0
